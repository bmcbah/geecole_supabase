# Catalogue — Types d’évaluation

Le catalogue est fourni par GeeCole et surchargeable. Il comprend notamment : interrogation, devoir, devoir surveillé, devoir à domicile, évaluation, composition, examen, examen blanc, oral, travaux pratiques, travaux dirigés, projet, exposé, contrôle continu et rattrapage.

L’établissement active les types utiles et configure leur libellé et leur barème. Le type ne porte pas de coefficient : son poids est défini dans chaque version de formule de calcul.
