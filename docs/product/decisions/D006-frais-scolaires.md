# D006 — Frais scolaires

Le paiement pendant l’inscription est configurable. L’établissement peut autoriser :

- une inscription sans paiement immédiat ;
- un acompte obligatoire ;
- un paiement total obligatoire ;
- un paiement facultatif effectué dans le stepper.

La même politique est appliquée côté interface et côté backend. Le dossier financier peut être créé avant, pendant ou immédiatement après la validation de l’inscription selon la politique choisie.
