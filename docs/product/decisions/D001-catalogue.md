# D001 — Catalogue et surcharges locales

GeeCole fournit les catalogues de cycles, niveaux, matières, périodes types, frais, documents administratifs, appréciations, profils et paramètres par défaut.

L’établissement peut :

- activer ou désactiver un élément ;
- appliquer un libellé local ;
- compléter les paramètres locaux ;
- ajouter une extension locale lorsque le catalogue ne couvre pas un besoin réel.

Le catalogue source reste immuable. Toute personnalisation est enregistrée comme surcharge locale liée à l’établissement, éventuellement à l’année scolaire.
