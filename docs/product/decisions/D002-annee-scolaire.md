# D002 — Année scolaire

Une année scolaire appartient à un établissement et possède l’un des états suivants :

- `draft` : préparation et configuration ;
- `active` : année courante ;
- `closed` : opérations ordinaires verrouillées ;
- `archived` : année conservée pour consultation historique.

Une seule année peut être active par établissement. Les cycles, niveaux, classes, périodes, matières et tarifs sont versionnés par année.
