# D004 — Évaluations et bulletins

Les types d’évaluation proviennent d’un catalogue fourni et surchargeable. L’établissement active ceux dont il a besoin et configure leurs paramètres.

Une évaluation est créée depuis un cours disponible. Les notes peuvent être saisies par classe ou individuellement, mais toujours dans le contexte d’une évaluation.

Les formules utilisent des variables correspondant aux types actifs. La valeur d’un type est la moyenne des notes de ce type pour l’élève, la matière et la période.

Les bulletins sont calculés, contrôlés puis publiés par lot. Une erreur sur un élève ne bloque pas les bulletins valides des autres élèves.
