# D005 — Documents

Le terme métier commun est « Document administratif » pour les pièces demandées ou importées dans un dossier.

La page profil de l’élève regroupe cependant plusieurs espaces : documents administratifs, bulletins, relevés de notes, certificats/attestations et autres documents générés.

Les modèles de documents proviennent d’un catalogue GeeCole surchargeable localement.
