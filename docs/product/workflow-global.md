# Workflow global

> **Statut : décision consolidée**  
> Cette documentation intègre les arbitrages validés le 20 juillet 2026.


## Mise en service

1. Création du workspace et de l’établissement.
2. Activation du compte administrateur.
3. Première connexion.
4. Choix ou import du catalogue.
5. Création de l’année scolaire en état `draft`.
6. Activation des cycles, niveaux, matières et périodes.
7. Configuration des classes ou activation du mode niveau = classe.
8. Création du personnel et des enseignants.
9. Configuration des affectations pédagogiques.
10. Activation de l’année scolaire.

## Parcours annuel

Configuration → inscriptions → affectations → cours disponibles → assiduité → évaluations → notes → bulletins → réinscriptions → clôture → archivage.

## Parcours élève

Liste des élèves → drawer d’aperçu → page profil complète.

L’inscription et la réinscription utilisent un stepper. Les actions rapides restent accessibles depuis le drawer.
