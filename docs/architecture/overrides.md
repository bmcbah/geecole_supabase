# Surcharges locales

## Principe

Le catalogue GeeCole est immuable. Un établissement personnalise un élément par une surcharge locale.

## Une surcharge peut contenir

- libellé local ;
- statut actif ;
- paramètres métier ;
- ordre d’affichage ;
- coefficient ou barème ;
- métadonnées propres à l’établissement ;
- portée annuelle lorsque nécessaire.

## Résolution

La valeur effective est calculée ainsi : surcharge annuelle → surcharge établissement → valeur catalogue.
