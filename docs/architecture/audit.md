# Audit et historique

Les opérations sensibles sont auditées : changement d’affectation, publication de bulletin, modification de note, encaissement, annulation, changement de tarif, clôture d’année et modification de permissions.

Chaque événement conserve : acteur, date, établissement, année, type d’action, identifiant métier, avant/après lorsque pertinent et motif pour les annulations.

Une donnée historique publiée n’est jamais remplacée silencieusement. Une correction crée une nouvelle version ou un événement d’ajustement.
