# Module Notes

Le module Notes couvre les affectations pédagogiques, les types de note, le cahier de notes, la saisie collective ou individuelle, les appréciations, les calculs et le suivi de complétude.

> Dans GeeCole, une **note** est l’activité notée créée dans le cahier.  
> Une **évaluation** est seulement un type de note, comme DS, examen, composition ou oral.
