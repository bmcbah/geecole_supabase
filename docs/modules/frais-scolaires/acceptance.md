# Critères d’acceptation — Frais scolaires

- Une inscription valide produit un dossier financier lorsque la politique le demande.
- Un paiement obligatoire bloque la finalisation tant que le minimum requis n’est pas encaissé.
- Un paiement facultatif peut être ignoré sans perdre le dossier.
- Une annulation ne supprime pas l’historique.
- La vue familiale agrège les élèves partageant au moins un parent.
