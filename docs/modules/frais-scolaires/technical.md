# Frais scolaires — Technique

Le module réagit aux événements d’inscription et crée ou prépare le dossier financier selon la politique de l’établissement.

Les agrégations familiales sont calculées à partir des liens parent–élève. Aucun identifiant Famille obligatoire n’est requis.

Les tarifs et plans sont versionnés. Les encaissements et annulations sont audités. Les reçus possèdent une numérotation stable définie par établissement.
