# Paramétrage — Spécifications UX

- Navigation latérale groupée par domaine.
- Sélecteur global d'année visible.
- Cycles et Niveaux sont deux rubriques distinctes.
- Cycles : cartes d'activation issues du catalogue Supabase.
- Niveaux : un onglet ordonné par cycle actif.
- Les états préparation, ouvert, clôturé et archivé sont toujours visibles.
- Toute action verrouillée explique sa cause.
- Les formulaires importants utilisent des panneaux ou dialogues courts.
- Les tables utilisent recherche, filtres, pagination et états vides explicites.
