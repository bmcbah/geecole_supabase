# Bulletins — Modèle technique

## Instantané

Chaque version conserve :

- résultats ;
- moyennes ;
- coefficients ;
- formules ;
- appréciations ;
- rang ;
- assiduité incluse ;
- modèle ;
- paramètres de rendu.

## Lot de génération

Un lot conserve contexte, périmètre, options, initiateur, progression et résultat par élève.

Chaque élève est traité indépendamment.

## Version

Une version publiée est immuable. Toute correction crée une nouvelle version.

## Validation

Conserver utilisateur, permission utilisée, date, version et commentaire éventuel.

## Publication

Conserver canaux, date, utilisateur, version et état de livraison.

## Rapport

Chaque anomalie possède un code, une sévérité, un message compréhensible et un lien vers la source à corriger.
