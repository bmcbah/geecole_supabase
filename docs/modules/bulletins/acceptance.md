# Bulletins — Critères d’acceptation

- Le périmètre peut être école, cycle, niveau, classe ou élève.
- Le formulaire est dynamique.
- Les contrôles distinguent prêts, avertissements et bloqués.
- Un élève bloqué n’empêche pas les autres.
- Reporté bloque le bulletin concerné.
- Le rapport propose un accès à la source.
- Une seule validation est nécessaire.
- Plusieurs profils peuvent valider.
- Publication et validation sont séparées.
- Un bulletin publié est immuable.
- Une correction crée une nouvelle version.
- Toutes les versions restent consultables et archivées.
