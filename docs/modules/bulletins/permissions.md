# Bulletins — Permissions

Permissions configurables :

- générer ;
- consulter les rapports ;
- valider ;
- publier ;
- consulter les versions ;
- configurer les modèles et options.

La permission de publication est distincte de la validation.

# Responsabilités de cycle

Les permissions Notes & Bulletins ne sont jamais déduites du seul intitulé professionnel. Elles sont accordées par rôle applicatif et peuvent être complétées par une responsabilité active sur le cycle concerné.

Le catalogue de responsabilités précise notamment si le titulaire peut piloter les périodes ou valider les bulletins. Le périmètre reste limité à l'année et au cycle de l'affectation.
