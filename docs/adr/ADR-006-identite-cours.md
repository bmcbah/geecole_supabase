# Identité du cours

## Statut
Accepté

## Décision
Le cours est identifié par année, classe/niveau, matière et période. L’enseignant est résolu par affectation et n’entre pas dans l’identité métier.

## Conséquences
La base de données, les services, la sécurité, les imports et l’interface doivent respecter cette décision.
