# Paiement configurable à l’inscription

## Statut
Accepté

## Décision
La politique peut rendre le paiement absent, facultatif, partiel obligatoire ou total obligatoire. Le front et le backend appliquent la même décision.

## Conséquences
La base de données, les services, la sécurité, les imports et l’interface doivent respecter cette décision.
