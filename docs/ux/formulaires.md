# Standards des formulaires

Les formulaires utilisent des sections cohérentes, une validation immédiate, des messages homogènes et l’autosave lorsque le risque est maîtrisé.

Les champs déjà déterminés par le contexte ne sont pas redemandés. Les listes dépendantes sont filtrées. Les champs libres ne contournent pas la configuration scolaire.
