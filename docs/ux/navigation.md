# Navigation

## Règle générale

`Liste → Drawer → Page profil complète`

- Le clic sur une ligne ouvre le drawer.
- Le drawer affiche un aperçu et les actions rapides.
- La page complète porte le profil, l’historique et les modifications importantes.
- Le stepper est réservé aux assistants structurés, notamment inscription, réinscription et mise en service.

La sidebar contient les modules. Le header contient le contexte workspace, établissement et année.
